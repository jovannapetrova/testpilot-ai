# TestPilot AI API

- `GET /api/health` - backend health
- `POST /api/analyze/upload` - upload ZIP and analyze
- `POST /api/analyze/github` - clone GitHub repository and analyze
- `GET /api/sample-projects` - list included samples
- `POST /api/analyze/sample/{sample_name}` - analyze sample project
- `GET /api/reports/{filename}` - download PDF/JSON report
